   <h2></h2>
   <div class="legend"><span style="">
         <p>Вася и Петя завели общий почтовый ящик и договорились, что паролем от него будет число <span class="tex-math-text">a</span> <span class="tex-math-text">mod</span> <span class="tex-math-text">b</span>. Пароль утверждал Петя, который использовал для подсчётов язык С++. Он забыл, что если <span class="tex-math-text">a</span> отрицательное, а <span class="tex-math-text">b</span> &mdash; положительное, то остаток от деления будет равен <span class="tex-math-inline"><img class="tex-math" src="/testsys/tex/render/LSAofGF8IFxibW9kIGIp.png"></span>, то есть будет отрицательным, вопреки определению (такой он загадочный, этот C++). Помогите питонисту Васе восстановить пароль.
         </p></span></div>
